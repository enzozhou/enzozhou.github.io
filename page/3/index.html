iPhone 5 睡眠/唤醒按钮更换计划

查询地址：https://ssl.apple.com/cn/support/iphone5-sleepwakebutton/

Apple 已确定一小部分 iPhone 5 机型上的睡眠/唤醒按钮装置可能会停止工作或间歇性工作。至 2013 年 3 月生产的 iPhone 5 机型都可能会受此问题影响。

Apple 将免费更换出现此问题且具有符合条件的序列号的 iPhone 5 机型上的睡眠/唤醒按钮装置。

睡眠/唤醒按钮位于 iPhone 5 顶部：
资格
如果您的睡眠/唤醒按钮无法正常工作，请在下面输入您的 iPhone 5 序列号，查看它是否符合此计划的条件。

请参阅“更换流程”部分，了解后续操作。

查找 iPhone 序列号非常容易。了解具体方式。

注：如果客户的 iPhone 5 睡眠/唤醒按钮没有表现出此问题的任何迹象且/或没有符合条件的序列号，则此时无需采取任何措施。
更换流程
睡眠/唤醒按钮装置的更换工作在 Apple 维修中心展开。请参阅下文，了解详情。从维修中心收到您的 iPhone 到退还给您，更换流程约需 1-2 周时间，具体取决于您所在的地点。在提供任何维修服务前，将对您的 iPhone 进行检查，以验证其符合该计划的条件，且除该问题之外可正常工作。在将 iPhone 寄送到 Apple 维修之前，您需要备份所有数据，并抹掉所有内容和设置。

注：如果您的 iPhone 5 存在屏幕开裂等任何损坏情况，影响睡眠/唤醒按钮的更换，需在睡眠/唤醒按钮更换前解决该问题。在某些情况下，可能需要支付维修费用。

如何将 iPhone 5 送到 Apple 维修中心

将您的 iPhone 5 带至参加此项计划的提供商- Apple Store 零售店或 Apple 授权服务提供商处。
将先对您的 iPhone 进行检查，以验证其符合条件，然后再将其送至当地 Apple 维修中心。
iPhone 可供取货时，将会通知您。

如果在 iPhone 维修期间，您需要一部备用机，参加计划的 Apple Store 零售店和大部分 Apple 授权服务提供商将提供 iPhone 5 机型（仅限 16GB）。 请与 Apple 授权服务提供商核实，以确认他们提供备用机。
注：如需送修，您必须前往 Apple Store 零售店或参加计划的 Apple 授权服务提供商，而非无线运营商合作伙伴。
准备进行 iPhone 5 维修
完成以下步骤，以确保您的 iPhone 可进行维修：

将您的数据备份至 iTunes 或 iCloud
关闭“查找我的 iPhone”
在“设置”>“通用”>“还原”>“抹掉所有内容和设置”中，抹掉数据和设置
取下所有保护套或保护壳

注：如果您的 iPhone 运行的是 iOS 6，将会在睡眠/唤醒按钮装置更换后升级至 iOS 7。
其他信息
Apple 可能会将维修区域限制为原购买国家/地区。

您的 iPhone 5 必须能够正常工作，才符合本计划的要求。

如果您确信已为针对此问题进行的更换支付了费用，请联系 Apple 咨询退款事宜。

这项全球范围的 Apple 计划不会延长 iPhone 的标准保修期限。

本计划涵盖在首次零售之后2年内的iPhone 5机型。

Ubuntu 下安装MySQL、Apache、PHP

  1. 任意位置打开终端。
  2. 安装MySQL
    运行 sudo apt-get install mysql-server mysql-client，
    等待弹出窗口提示输入MySQL root 密码，需要输入2次，
    安装完成后需要测试安装成功 运行 mysql uroot -p 看到欢迎信息 “Welcome to the MySQL monitor. ”表示安装好了。
  3. 安装Apache
    运行 sudo apt-get install apache2 ,
    安装完成后测试输入http://localhost或http://127.0.0.1 出现“It works!”信息表示安装成功。

It works!

  1. 安装PHP 并让Apache支持PHP
    运行 sudo apt-get install php5 libapache2-mod-php5

安装完成后重启Apache服务器
sudo /etc/init.d/apache2 restart
然后进入Apache的www文件夹下(默认在/var里面)，创建phpinfo.php文件
cd /var/www
ls
vi phpinfo.php
然后在phpinfo.php文件里写入
<?php
phpinfo();
?>
运行:w可能无法保存。
www目录没有写权限，所以需要对www目录赋予写权限
运行 sudo chmod 777 /var/www 提示输入用户密码，www目录写权限有了，可以保存phpinfo.php文件了。
浏览器打开http://localhost/phpinfo.php就可以看到php信息了。

  1. 安装的php-mysql软件包以及其它的一些常用的模块 sudo apt-get install php5-mysql php5-curl php5-gd php5-idn php-pear php5-imagick php5-imap php5-mcrypt php5-memcache php5-ming php5-ps php5-pspell php5-recode php5-snmp php5-sqlite php5-tidy php5-xmlrpc php5-xsl 安装好之后需要重启Apache sudo /etc/init.d/apache2 restart，重启完就可一看到安装的模块了。
  2. 安装phpMyAdmin
    通过phpMyAdmin可以很方便的管理我们的MySQL数据库
    sudo apt-get install phpmyadmin 安装过程中会问是否要配置phpmyadmin，我们选择“YES”
    然后需要输入MySQL的密码，直至安装完成。
    由于默认情况下phpMyAdmin的安装路径为/usr/share/phpmyadmin/，那么我们把phpmyadmin移动到/var/www/目录下：
    sudo mv /usr/share/phpmyadmin /var/www
    然后http://localhost/phpmyadmin访问，就到了phpMyAdmin的登陆界面_
